# ♟️ Desafio Xadrez em C#

Este projeto valida movimentos de peças do xadrez via terminal, feito em C# para Ubuntu.

## 🧠 Peças suportadas

- Torre (linha ou coluna)
- Bispo (diagonal)
- Cavalo (em L)

## ▶️ Como executar

### Requisitos
- .NET SDK 8 ou superior

### Passos

```bash
dotnet run --project Xadrez
